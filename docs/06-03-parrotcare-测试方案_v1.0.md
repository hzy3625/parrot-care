## 一、测试概述

### 1.1 测试目标

确保 ParrotCare AI 鹦鹉健康监测系统的核心功能正确性、性能和用户体验满足需求规格。

### 1.2 测试范围

| 模块 | 测试类型 | 优先级 |
|------|---------|--------|
| 用户注册/登录 | API 测试、UI 测试 | P0 |
| 鹦鹉档案管理 | API 测试、UI 测试 | P0 |
| 音频上传与分类 | API 测试、集成测试 | P0 |
| 异常事件检测 | 集成测试、边界测试 | P0 |
| 健康评分计算 | 单元测试、集成测试 | P1 |
| 实时分析推送 | 集成测试 | P1 |
| 用户反馈 | API 测试、UI 测试 | P2 |

### 1.3 测试环境

| 环境 | 配置 |
|------|------|
| 操作系统 | Windows 10 / Linux |
| Python | 3.10+ |
| 数据库 | SQLite (本地测试) |
| 浏览器 | Chrome/Edge (Web UI) |
| 测试工具 | pytest + requests + Playwright |
